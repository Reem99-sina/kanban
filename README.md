Kanban Board - Next.js + React + DnD Kit

A responsive Kanban board built with Next.js, React, MUI, and @dnd-kit/core, featuring:

    Drag-and-drop tasks between columns
    Add, edit, and delete tasks
    Search tasks by title or description
    Persistent state using a mock API (JSON or REST API)

Features
    Columns: Backlog, In-Progress, Review, Done
    Task Actions: Add, Edit, Delete
    Drag & Drop: Reorder tasks within columns or move across columns
    Search: Filter tasks by title or description in real-time
    Responsive UI: Grid layout with MUI components

Getting Started
1. Install Dependencies
    npm install
# or
    yarn install
2. Start Development Server
    npm run dev
# or
    yarn dev

Open http://localhost:3000
 in your browser.
